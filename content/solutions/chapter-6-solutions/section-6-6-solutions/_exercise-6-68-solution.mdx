a) Using the results from Exercise 6.67,

$$
\begin{align*}
    f_{U_1, U_2}(u_1, u_2) & = f_{Y_1, Y_2}(u_1 u_2, u_2) |u_2|\\
    & = |u_2|
    \begin{cases}
        8(u_1 u_2)(u_2), & 0 \leq u_1 u_2 < u_2 \leq 1,\\
        0, & \text{elsewhere,}
    \end{cases}\\
    & =
    \begin{cases}
        8u_1 u_2^3, & 0 \leq u_1 < 1, 0 < u_2 \leq 1,\\
        0, & \text{elsewhere.}
    \end{cases}
\end{align*}
$$

Note that $0 \leq u_1 u_2 < u_2 \leq 1 \Longleftrightarrow 0 \leq u_1 < 1 \text{ and } 0 < u_2 \leq 1$ because

$$
\begin{align*}
    0 \leq u_1 \text{ and } 0 < u_2 & \Longleftrightarrow 0 = 0 \cdot 0 \leq u_1 \cdot u_2 = u_1 u_2 \text{ and } 0 < u_2,\\
    u_1 < 1 \text{ and } 0 < u_2 & \Longleftrightarrow u_1 u_2 = u_1 \cdot u_2 < 1 \cdot u_2 = u_2.
\end{align*}
$$

b)

$$
\begin{align*}
    f_{U_1}(u_1) & =
    \begin{cases}
        \int_{u_2 = 0}^{u_2 = 1} 8u_1 u_2^3 du_2\\
        = 2u_1 u_2^4 \biggr|_{u_2 = 0}^{u_2 = 1}\\
        = 2u_1 (1)^4 - 2u_1 (0)^4\\
        = 2u_1, & 0 \leq u_1 < 1,\\
        0, & \text{elsewhere}
    \end{cases}\\
    \text{and } f_{U_2}(u_2) & =
    \begin{cases}
        \int_{u_1 = 0}^{u_1 = 1} 8u_1 u_2^3 du_1\\
        = 4u_1^2 u_2^3 \biggr|_{u_1 = 0}^{u_1 = 1}\\
        = 4(1)^2 u_2^3 - 4(0)^2 u_2^3\\
        = 4u_2^3, & 0 < u_2 \leq 1,\\
        0, & \text{elsewhere}
    \end{cases}\\
    \implies f_{U_1}(u_1) f_{U_2}(u_2) & =
    \begin{cases}
        2u_1, & 0 \leq u_1 < 1,\\
        0, & \text{elsewhere}
    \end{cases} \cdot
    \begin{cases}
        4u_2^3, & 0 < u_2 \leq 1,\\
        0, & \text{elsewhere}
    \end{cases}\\
    & =
    \begin{cases}
        8u_1 u_2^3, & 0 \leq u_1 < 1, 0 < u_2 \leq 1,\\
        0, & \text{elsewhere}
    \end{cases}\\
    & = f_{U_1, U_2}(u_1, u_2)
\end{align*}
$$

Thus, $U_1$ and $U_2$ are independent.
