a)

$$
f_{U_1}(u_1) =
\begin{cases}
    \int_{u_2 = -\infty}^{u_2 = \infty} f_{U_1, U_2}(u_1, u_2) du_2\\
    = \int_{u_2 = 0}^{u_2 = \infty} \frac{1}{\beta^2} u_2 e^{-u_2 / \beta} du_2\\
    = \int_{u_2 = 0}^{u_2 = \infty} \frac{u_2^{2 - 1} e^{-u_2 / \beta}}{\beta^2 \Gamma(2)} du_2\\
    = 1 = \frac{1}{1 - 0}, & 0 < u_1 < 1,\\
    0, & \text{elsewhere.}
\end{cases}
$$

This is exactly the density function for a uniform random variable over $(0, 1)$.

---

b)

$$
f_{U_2}(u_2) =
\begin{cases}
    \int_{u_1 = 0}^{u_1 = 1} \frac{1}{\beta^2} u_2 e^{-u_2 / \beta} du_1\\
    = \frac{1}{\beta^2} u_2 e^{-u_2 / \beta} u_1 \biggr|_{u_1 = 1}^{u_1 = 0}\\
    = \frac{1}{\beta^2} u_2 e^{-u_2 / \beta} (1) - \frac{1}{\beta^2} u_2 e^{-u_2 / \beta} (0)\\
    = \frac{1}{\beta^2} u_2 e^{-u_2 / \beta} = \frac{u_2^{2 - 1} e^{-u_2 / \beta}}{\beta^2 \Gamma(2)}, & 0 < u_2,\\
    0, & \text{elsewhere.}
\end{cases}
$$

This is exactly the density function for a gamma random variable with parameters $\alpha = 2$ and $\beta$.

---

c)

$$
\begin{align*}
    f_{U_1}(u_1) f_{U_2}(u_2) & =
    \begin{cases}
        1, & 0 < u_1 < 1,\\
        0, & \text{elsewhere}
    \end{cases} \cdot
    \begin{cases}
        \frac{1}{\beta^2} u_2 e^{-u_2 / \beta}, & 0 < u_2,\\
        0, & \text{elsewhere}
    \end{cases}\\
    & =
    \begin{cases}
        \frac{1}{\beta^2} u_2 e^{-u_2 / \beta}, & 0 < u_1 < 1, 0 < u_2,\\
        0, & \text{elsewhere}
    \end{cases}\\
    & = f_{U_1, U_2}(u_1, u_2)
\end{align*}
$$

Thus, $U_1$ and $U_2$ are independent by Theorem 5.4.
