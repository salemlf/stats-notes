## Callouts

More documentation about Callouts can be found [here](https://fumadocs.vercel.app/docs/ui/components/callout).

### Info

This is the default callout type.

**Syntax**

```mdx
<Callout title="Info Callout">Content here</Callout>
```

**Example**

<Callout title="Info Callout">Content here</Callout>

### Tip

**Syntax**

```mdx
<Callout title="Tip Callout" type="tip">
  Content here
</Callout>
```

**Example**

<Callout title="Tip Callout" type="tip">

</Callout>

## Code Blocks

### JavaScript Code Block

```js
console.log("Hello World");
```

```md

- Item #1
- Item #2
```

## Cards

**Syntax**

```mdx
<Cards>
  <Card title="Learn more about Next.js" href="https://nextjs.org/docs" />
  <Card title="Learn more about Fumadocs" href="https://fumadocs.vercel.app" />
</Cards>
```

**Example**

<Cards>
  <Card title="Learn more about Next.js" href="https://nextjs.org/docs" />
  <Card title="Learn more about Fumadocs" href="https://fumadocs.vercel.app" />
</Cards>
