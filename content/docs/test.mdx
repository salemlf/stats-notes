## Accordions

More documentation about Accordions can be found [here](https://fumadocs.vercel.app/docs/ui/components/accordion).

**Syntax**

```mdx
<Accordions>
  <Accordion title="My Title">My Content</Accordion>
</Accordions>
```

**Example**

<Accordions>
  <Accordion title="My Title">My Content</Accordion>
</Accordions>

## Callouts

More documentation about Callouts can be found [here](https://fumadocs.vercel.app/docs/ui/components/callout).

### Info Callout

This is the default callout type.

**Syntax**

```mdx
<Callout title="Info Callout">Content here</Callout>
```

**Example**

<Callout title="Info Callout">Content here</Callout>

### Warning Callout

**Syntax**

```mdx
<Callout title="Warning Callout" type="warn">
  Content here
</Callout>
```

**Example**

<Callout title="Info Callout" type="warn">
  Content here
</Callout>

### Error Callout

**Syntax**

```mdx
<Callout title="Error Callout" type="error">
  Content here
</Callout>
```

**Example**

<Callout title="Error Callout" type="error">
  Content here
</Callout>

## Cards

**Syntax**

```mdx
<Cards>
  <Card title="Learn more about Next.js" href="https://nextjs.org/docs" />
  <Card title="Learn more about Fumadocs" href="https://fumadocs.vercel.app" />
</Cards>
```

**Example**

<Cards>
  <Card title="Learn more about Next.js" href="https://nextjs.org/docs" />
  <Card title="Learn more about Fumadocs" href="https://fumadocs.vercel.app" />
</Cards>
